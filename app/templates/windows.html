Installation Guide

Quick and easy setup for Windows

1

Download the Installer

Choose either the standard installer or portable version above.

2

Run the Installer

  • Double-click the downloaded .exe file
  • If Windows SmartScreen appears, click "More info" then "Run anyway"
  • Follow the installation wizard prompts
  • Choose installation location (default is recommended)
3

Launch QuickStream

  • Find QuickStream in your Start Menu or on your desktop
  • Double-click to launch
  • Start streaming for free!
💡

Portable Version Instructions

  • Extract the downloaded ZIP file to your desired folder
  • Run QuickStream.exe directly
  • No installation or admin rights required
  • Your settings are saved in the same folder
Security Note: QuickStream is completely safe and does not contain any malware. If Windows SmartScreen appears, it's because the app is new - you can safely install by clicking "More info" then "Run anyway".
